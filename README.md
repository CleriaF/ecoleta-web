## Como rodar a aplicação :arrow_forward:

## Pré-requisitos

- [Git](https://git-scm.com/)
- [Node](https://nodejs.org/en/download/)

No terminal, clone o projeto: 

```
git clone https://github.com/CleriaF/ecoleta-web.git
```

Baixe as dependências:

```
yarn install
```

Execute a aplicação:

```
yarn start
```
